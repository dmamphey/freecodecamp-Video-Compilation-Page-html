# Video Compilation Page

A simple HTML project built as part of my coding practice.

## About the Project

This project is a basic video compilation webpage focused on **Front-End Web Development**. It introduces three core front-end technologies, HTML, CSS, and JavaScript, and includes embedded tutorial videos for each topic.

The page is designed to help practise structuring educational content on a webpage using HTML.

## Features

- Main heading and introductory description
- Separate sections for HTML, CSS, and JavaScript
- Embedded YouTube tutorial videos using `iframe`
- Informational text for each front-end topic
- Clear content structure using sections and headings

## Built With

- HTML5

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Structuring a page with headings, paragraphs, and sections
- Embedding external videos using the `iframe` element
- Using attributes such as `title`, `allow`, `referrerpolicy`, and `allowfullscreen`
- Organising educational content clearly on a webpage
- Building a simple multi-section layout
